Simple BURN
1. Alice CREATEs a collection
2. Alice MINTs an NFT
3. Alice BURNs NFT

BURN grandchild
1. Alice CREATEs a collection
2. Alice MINTs an NFT
3. Alice MINTs another NFT
4. Alice SENDs NFT2 to NFT1 (NFT2 becomes child of NFT1)
5. Alice BURNs NFT2

BURN parent
1. Alice CREATEs a collection
2. Alice MINTs an NFT
3. Alice MINTs another NFT
4. Alice SENDs NFT2 to NFT1 (NFT2 becomes child of NFT1)
5. Alice BURNs NFT1 (NFT2 gets BURNed as well)
